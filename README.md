### 1. Student Intake
Collect, validate and normalize attendance from SIS; attach a runId and timestamp for traceability.

### 2. Draft
Execute draft phase for the Reflection pattern: persist interim state, enforce guardrails, and emit structured JSON results.

### 3. Self-Critique
